发生认识论原理



皮亚杰


序言

  • 在认识论哲学中居支配地位的认识论哲学家们,都是从 逻辑分析和 语言分析 出发.而不是从心理学分析出发.
  • 发生认识论==经验论的发展
  • 每一个结构都是心理发生的结果,而心理发生就是从一个较初级的结构过渡到一个不那么初级的结构.
  • 所有认识都包含有新东西的加工制作的一面.
  • 布隆施维克: 推理 本质上是动态的认识论
  • 建立一个可以提供经验验证的方法;追溯认识的起源
  • 一般来说,发生认识论的特有问题是认识的成长问题:
    • 从一种不充分的比较贫乏的认识,向在深度广度上都较为丰富的认识的过渡
  • 把认识看作是一种继续不断的建构

认识的形成;认识的心理发生

  • 传统经验主义假定:主体是受教与他以外之物的
  • 先验主义和天赋论则认为:主体一开始就具有一些内部生成的结构;并把这些结构强加于客体
  • 心理发生学分析的初步结果: 认识起源于主客体之间的相互作用.
  • 一开始起中介作用的并不是知觉,而是可塑性大的多的活动本身
  • 从动作转变到概念化思维

    幼儿没有显示出任何自我意识,也不能在内部给予的东西和外部给予的东西之间做出固定不变的划分;[自我中心化,无意识的]

  • 任何两种活动取得协调的前提是主动性.
  • 主体的身体开始被看作是处于一个空间中的诸多客体中的一个.
  • 经验主义心理学的中心概念是 联想
  • 结构就其从遗传的来而言是预先就存在的:吮吸反射;第二种是未曾预料到的
  • 这种概念在表征活动时,能超越于这些活动之上
  • 活动的内化就是概念化

    儿童完全熟悉从家里到学校的路,但却不能在实验中用已有的里程标志来描述这条道路{还没有抽象化}

  • 表象思维->表象系统->超时空呈现
  • 聋哑儿童具有与正常儿童相似的认知结构
  • 模仿活动内华为表象作用的形式
  • 创新行为的发展形式具有一定的限度
  • 自我中心化->分化{解除自我中心化}
  • 同一性:从一些可观察的质中把把那些没有变化的质同那些改变了的质区分开来
  • 量的守恒的先决条件则是 新关系 构建
  • A< B;B < C ;===> A< C
  • 绝对的开始在发展过程中是永远开不到的.
  • 新的东西是逐步的分化和渐进的协调的结果
  • 可逆性,系统的闭合性
  • 数表现为归类运演和序列化运演相融合
  • 类\关系\数\ 假设->推理\依存关系\超时间性{形式运演}
  • 逻辑数学运演的发展影响着因果关系的发展
  • 从出生以后,就一直在活跃的进行着的内化和外化的平行发展

原初的有机体条件:认识的生物发生

  • 拉马克的经验主义:变异和演化学说
    • 起源于外界的学习假说
  • 休谟试图用习惯和联想的机制来说明心理事实
  • 学习过程就是获得成就的基本过程
  • 学习在发展的不同水平上是不同的
  • 天赋论:乔姆斯基-"生成语法的转换规则最终将揭示出一种固定内核"
    • 生物学上看,就要证明:大脑中枢预先就具有言语和理性思维的基本形式.
    • 即:智力是语言的基础,而不是语言是智力的基础
    • 认识的结构同样是天赋的{生态学家洛伦兹}
  • 我们的解释:认识的结构确实是赢得了必然性的,但是只是在他们发展的 最后 ,而不是一开始就有,而且也不牵涉任何先行的遗传程序设计
  • 遗传和成熟的作用都限于:只能决定后天成就的不可能性或者说可能性的范围有多大.但是,成就的实现需要由经验从而也是由环境所给与的外界材料,以及由自我调节引起的逐步的内部组织化.
  • 自我调节的三个特点:
    • 自我调节是遗传特性传递下去的先行条件
    • 自我调节比遗传特性的传递更为普遍
    • 自我调节最后导致高级水平的必然性的出现
  • 遗传特性并不是一个必然要遗传下去的问题
  • 调节作用从一开始就需要在常态和变态之间做出区分;倾向于更为重视常态特性

从本能到智力

  • 本能包含有行为内容和行为形式的遗传程序编制
  • 了解从本能到智力的过度;
  • 智力从本能之中出现是伴随着不同但相对联系着的两种发展
    • 一种是内化的,是朝向逻辑数学方向
    • 另一种是外化的,是朝向学习和经验
  • 原始遗传组织与环境的相互作用
  • 自我调节系统->稳态系统
  • 自我调节是生命最普遍的特征之一
  • 以自我调节为基础的解释特别富有成效,因为这些解释说明了形成结构的过程
  • 有机体的自我调节以给我们提供了从一个水平到另一个水平的无限重建过程的图景,不是高级形式就预先包含在低级形式之中,而是高低级形式之间的联结仅仅是功能上的类似
  • 由不断的自我调节的功能作用所产生的结构的连续性
  • 从神经元逻辑到思维水平的命题逻辑:需要11~12年的反身抽象的建构作用

古典认识论问题的重新考虑

逻辑的认识论

  • 三个基本问题:
    • 形式化的程序和"自然"思维的程序之间有什么联系?
    • 被逻辑化了的是什么?
    • 为什么形式化会有戈德尔意义下的局限性?
  • 数学家帕施:形式化的程序和自然思维的自发倾向是背道而驰的
    • 一般思维倾向于向前看,而形式化却总是回顾过去
  • 三值逻辑:不同却十分接近于一般思维的逻辑
    • 排中律：生存，还是毁灭，没有中间状态
    • 内涵是定义，外延是在符合此定义的东西
    • 同一律：同一的是外延与内涵。
  • 假定1:逻辑就是我们对客体认识的公理化
  • 假定2:逻辑是把自然思维形式化
  • 把逻辑看作是一种语言,逻辑就不是词本来意义下所指的认识的一种形式,而是认识的纯粹形式
  • 智力是先于言语而存在的;与活动格局协调(联合,归类,顺序,对应)
  • 人们发现 分析性判断 和 综合性判断 之间是有 中间情况 的,一切关系开始时都是综合性的,在某些情况下根据他们的内涵才变成分析性的
  • 尽管形式化有其独立性和威力,但现在已经证明其有确定不移的局限性
  • 存在着把结构按其"强度"排列的阶梯式体系
  • 结构 比作其高度在不断增加的螺旋体
  • 形式与内容的概念在本质上是相对的
  • 形式必然是会有局限性的
  • 一一对应的关系和拓扑结构在儿童身上是可以考察到的
  • 一种创造暗示着某种程度的自由
  • 丰富性和必然性总是连在一起的
  • 事件的 恒常性 和 因果性 之间的关系问题对认识论是有深远的影响的
  • 因果解释要求客体是实际存在的
  • 客观性是作为一种过程,而不是作为一种状态开始的

  • 现实性和可能性之间的关系问题
  • 我们的宇宙本身也是不完善的
  • 是创新还是预先形成的问题
  • 现实性被插在可能性与必然性之间
  • 一个体系的无矛盾性的论证是以另一个更强体系的建构为前提
  • 高级的东西可以借助转换从低级的东西中演化出来
  • 高级的东西可以通过整合低级的东西使低级的东西更丰富
  • 不同水平的两个结构之间不能有单向的还原
  • 认识的原初形式到高级形式的差别比我们认识的要大的多
  • 高级形式的建构不得不经历一段比人们所想象的更长的多\更困难\更不客预测的过程

想法

  • 当能够认识一个完整的系统时,你已经离开了这个系统{超越性}
  • 记忆:
    • 时间性记忆:一部电影回放,一段音乐回忆,一部小说
    • 空间性记忆:行政区划记忆
  • {学习的目标不是把自己变成他国，而是要把自己变得更好、更像自己}
  • {工业化时期形成的教育体系:产业化机械化专业细分批量生产}
  • 定理,概念,定义是什么?基本概念的组合?`
  • 历史形成的偶然性与必然性?{建构过程的概率}
  • 存在&&意识
    • 一张纸上有两个点{客观存在}->将两个点的距离定义为1分米;
    • 定义一分米的1/10为1cm{意识}->在两点之间等距离填满8个点形成一把尺子{意识发作用于存在}
